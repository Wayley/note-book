### 数据类型和结构类型

- 6 种数据类型(Data Type)，这一类数据可以使用`typeof`运算符检查

  - `Undefined`
  - `Boolean`
  - `Number`
  - `String`
  - `Symbol`
  - `BigInt`

- 2 种结构类型(Structural Type)

  - `Object`
  - `Function`

- 1 种结构根基元(Structural Root Primitive)
  - `Null`
